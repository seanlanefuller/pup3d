# PUP3D

PUP3D is a retro-style First-Person Shooter (FPS) built with HTML5 Canvas and JavaScript. Inspired by classics like Wolfenstein 3D, it features 3D raycasting rendering, maze-like levels, and classic FPS gameplay mechanics.

## Features

- **Retro 3D Rendering**: Classic raycasting engine for a nostalgic look and feel.
- **Classic Gameplay**: Navigate through mazes, collect colored keys to unlock doors, and find the exit.
- **Combat**: Eliminate enemies using your weapon while managing health and ammo.
- **Sound Effects**: Immersive audio for actions like shooting, opening doors, and collecting items.
- **Multiple Levels**: Progress through increasingly challenging environments.

## Controls

- **Move**: `W`, `A`, `S`, `D`
- **Look Around**: Mouse
- **Shoot**: Left Click
- **Interact / Open Doors**: `E`
- **Start / Restart**: On-screen buttons

## How to Play

1. Open `index.html` in any modern web browser.
2. Click **START GAME** on the title screen.
3. Defeat enemies, collect keys (Red, Blue, Yellow), and reach the level exit.
4. Try to survive and complete all levels!

## Development

The project is built using:
- **HTML5 Canvas**: For high-performance 2D-based 3D rendering.
- **Vanilla JavaScript**: Core game logic, physics, and state management.
- **CSS3**: For the HUD and screen layouts.
